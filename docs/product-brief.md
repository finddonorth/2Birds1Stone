# 2Birds1Stone Product Brief

## Core promise

2Birds1Stone helps creative, high-idea-volume users capture thoughts without losing focus. The product reduces mental clutter by giving every idea or task a trusted home, then turning the backlog into a system that can be reviewed calmly and acted on in order.

## Target user

- Creative builders with many parallel ideas
- Users with ADHD or attention-fragmented workflows
- Founders, executives, creators, and operators balancing goals, projects, and obligations

## Primary jobs to be done

1. Capture an idea or task the moment it appears
2. Organize it without heavy effort
3. Know what deserves attention now
4. Defer lower-priority work without fear of losing it
5. Review ideas later from phone or computer

## V1 feature set

### Setup

- Overall goals ranked in priority order
- Projects ranked in priority order
- Topics ranked in priority order
- Priority guidance text that instructs AI sorting
- Editable relative priority labels:
  - P1: Today
  - P2: End of Day
  - P3: End of Week
  - P4: Parking Lot
- Predefined tag library
- Notification defaults

### Capture

- Entry type: idea or task
- Voice or text input
- Fields: title, notes, tags
- Optional attachment: image, sketch, file, document
- Optional project and topic
- Relative priority selection
- Optional multi-day task toggle
- Optional start date and due date
- Optional follow-up contact with clickable phone/email

### Organize

- AI suggests:
  - title cleanup
  - project/topic assignment
  - tags
  - priority
  - whether item is better treated as a task or idea
- Manual edits always override AI

### Review surfaces

- To Do page
  - Relative Priority lane for P1-P4 tasks
  - Deadline lane for due-date-driven tasks
- Ideas to Tackle page
  - prioritized idea backlog
- Setup page
  - manage goals, projects, topics, tags, AI guidance, notification rules

### Notifications

- P1: every hour until complete
- P2: daily at 4:00 PM
- P3: Friday at 8:00 AM
- Due date reminders:
  - 3:00 PM day before
  - 9:00 AM day of
  - 3:00 PM day of

## Product behavior rules

- If a task has a due date, it appears in the deadline view
- If a task only has relative priority, it appears in the relative priority view
- Users can manually reorder lists; manual rank overrides AI rank
- Lower-priority items can be lightly gated if unfinished P1 items exist
- Sync should feel close to real time across devices

## Future ideas

- Shared workspaces for teams, assistants, or spouses
- Email and calendar integration
- Contact follow-up workflows
- Voice-first capture from watch or home screen widget
- Weekly review assistant
